`EXT_PLUGIN_FAILED` — A plugin setup or one of its hooks threw an error.

## When you see this

A user-supplied or first-party plugin raised an exception during the build.

## How to fix

The error message names the plugin and the hook. Look at that plugin source. If it is a third-party plugin, file an issue against it.
