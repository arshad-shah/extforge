`EXT_MANIFEST_INVALID` — The generated manifest fails MV3 validation.

## When you see this

A required field is missing, or a field has a value the browser rejects.

## How to fix

Run extforge validate to see the detailed errors. Common causes: missing icons, malformed content_scripts, unknown permissions.
